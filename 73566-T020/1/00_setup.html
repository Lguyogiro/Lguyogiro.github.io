Empezando

Windows 10

Sí tienes Windows 10, puedes instalar Windows Subsystem for Linux (WSL) para tener acceso a una terminal de Linux. Esto te permitirá usar Python y otros programas de la misma manera que en Linux.

Para la plataforma Linux, una buena opción es Ubuntu o Debian. Una vez que hayas instalado WSL, puedes abrir la terminal de Linux con el comando wsl en la línea de comandos de Windows. Para acceder a tus archivos de Windows desde la terminal de Linux, puedes ir a la carpeta /mnt/c/ y encontrarás tus archivos de Windows en la carpeta /mnt/c/Users/tu_usuario/.

Instalando Python con un entorno virtual

Yo recomiendo que usen un entorno virtual para Python, para evitar complicaciones y detalles en el futuro respecto a las diferentes versiones de Python y/o problemas con dependencias.

Quizás la opción menos complicada es pyenv (abre ese enlace y busca las instrucciones para tu sistema operativo). Una vez instalada, puedes ver todas las versiones de Python disponibles para instalar con: 

pyenv install --list

Para instalar una versión de Python, por ejemplo 3.12.8, puedes hacer:

pyenv install 3.12.8

Para crear un entorno virtual con esa versión de Python, puedes hacer:

pyenv virtualenv 3.12.8 mi_entorno
