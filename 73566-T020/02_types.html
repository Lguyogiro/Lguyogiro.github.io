Tipos básicos en Python

En Python, hay varios tipos de datos que puedes utilizar para almacenar y manipular información. A continuación, presento algunos de los más comunes.

Cadenas (string)

Las cadenas son secuencias de caracteres que se pueden almacenar en variables. Se definen utilizando comillas simples o dobles.

nombre = "Juan"
edad = '25'
print(nombre)  # Salida: Juan
print(edad)   # Salida: 25

Números decimales (float)

Los números decimales son números que tienen una parte decimal. Se pueden representar utilizando el tipo float.

altura = 1.75
peso = 70.5
print(altura)  # Salida: 1.75
print(peso)    # Salida: 70.5

Números enteros (int)

Los números enteros son números sin parte decimal. Se pueden representar utilizando el tipo int.

edad = 25
año = 2024
print(edad)   # Salida: 25
print(año)    # Salida: 2024

Listas (list)

Las listas son colecciones de elementos que se pueden almacenar en una variable. Se definen utilizando corchetes [] y se separan con comas.

frutas = ["manzana", "banana", "naranja"]
print(frutas)  # Salida: ['manzana', 'banana', 'naranja']

Si tienes un archivo de textos y quieres segmentarlo en oraciones, puedes almacenarlas en una lista. También son útiles las listas cuando queremos segmentas una frase en palabras.


texto = "Este es un ejemplo de texto en Python. Este es otro ejemplo."
oraciones = texto.split(".")
print(oraciones)  # Salida: ['Este es un ejemplo de texto en Python', ' Este es otro ejemplo', '']

palabras = oraciones[0].split(" ")
print(palabras)  # Salida: ['Este', 'es', 'un', 'ejemplo', 'de', 'texto', 'en', 'Python']

Diccionarios (Dict)

Los diccionarios son colecciones de pares clave-valor que se pueden almacenar en una variable. Se definen utilizando llaves {} y se separan con comas.

persona = {"nombre": "Juan", "edad": 25}
print(persona)  # Salida: {'nombre': 'Juan', 'edad': 25}

Conjuntos (Set)

Los conjuntos son colecciones de elementos únicos que se pueden almacenar en una variable. Se definen utilizando llaves {} y se separan con comas.

colores = {"rojo", "azul", "verde"}
print("rojo" in colores)  # Salida: True

Booleanos (Bool)

Los booleanos son valores que representan verdadero o falso. Se pueden representar utilizando los valores True y False.
