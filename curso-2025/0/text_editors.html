Escogiendo un editor de texto y/o EDI

Para poder escribir código, vas a necesitar un editor de texto. Hay muchas opciones. Aquí dejo unas sugerencias:

Editores de texto basado en la terminal:

  • Nanoeditor: un editor muy sencillo y popular en Linux.
  • Vim: Un editor de texto muy poderoso, pero con una curva de aprendizaje pronunciada. (Juego para aprender Vim)
  • Emacs: Otro editor de texto muy poderoso, pero con una curva de aprendizaje pronunciada.

Un editor popular que no está basado en la terminal:

  • Sublime Text: Un editor de texto muy popular y poderoso.

EDIs (Entornos de Desarrollo Integrados) son mucho más poderosos y son útiles para hacer proyectos más grandes:

  • Visual Studio Code: Un editor de texto muy popular y poderoso.
  • PyCharm: Un EDI muy popular para Python.
  • Eclipse: Un EDI muy popular para Java.
