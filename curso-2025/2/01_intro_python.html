Uso del intérprete

Antes de escribir programas en Python, vamos a abrir y usar el intérprete. Puedes abrirlo desde la línea de comandos con el comando python. Por ejemplo:

python

Una vez que estés dentro del intérprete, puedes escribir comandos y ver los resultados inmediatamente.

Imprimir texto

Una de las cosas más básicas que puedes hacer en Python es imprimir texto en la pantalla. Para hacer esto, utilizas la función print(). Por ejemplo:

print("Hola, mundo")

Esto imprimirá "Hola, mundo" en la pantalla.

Variables simples

En Python, puedes almacenar valores en variables. Una variable es un nombre que se asigna a un valor. Por ejemplo:

nombre = "Robert"
edad = 36
print("Mi nombre es", nombre, "y tengo", edad, "años")

Esto imprimirá "Mi nombre es Juan y tengo 25 años". Puedes cambiar el valor de una variable en cualquier momento.

Lectura y escritura de archivos de texto

Python también te permite leer y escribir archivos de texto con facilidad. Por ejemplo, para leer el contenido de un archivo llamado example.txt, puedes usar la función open():

archivo = open("example.txt", "r")
contenido = archivo.read()
print(contenido)
archivo.close()

Esto abrirá el archivo example.txt en modo lectura ("r"), leerá su contenido y lo imprimirá en la pantalla. Luego, cierra el archivo con close().

Para escribir texto en un archivo, puedes usar la función open() con el modo "w":

archivo = open("example.txt", "w")
archivo.write("Este es un ejemplo de texto escrito en Python")
archivo.close()

Esto creará un nuevo archivo llamado example.txt y escribirá el texto "Este es un ejemplo de texto escrito en Python" en él.

Corriendo un programa en Python

Para correr un programa en Python, tienes que escribir el código en un archivo con extensión .py y luego correrlo desde la línea de comandos usando el comando python "programa".py. Por ejemplo, si guardas el código anterior en un archivo llamado mi_programa.py, puedes correrlo con el comando:

python mi_programa.py
