Tipos básicos en Python

En Python, hay varios tipos de datos que puedes utilizar para almacenar y manipular información. A continuación, presento algunos de los más comunes.

Con cualquier valor, si quieres saber qué tipo de dato es, puedes usar la función type(). Por ejemplo:

        
        nombre = "Robert"
        print(type(nombre))  # Salida: <class 'str'>

También puedes usar la función isinstance() para verificar si un valor es de un tipo de dato específico. Por ejemplo:


edad = 25
print(isinstance(edad, int))  # Salida: True
print(isinstance(edad, str))  # Salida: False

Números enteros (int)

Los números enteros son números sin parte decimal. Se pueden representar utilizando el tipo int.


edad = 25
año = 2024
print(edad)   # Salida: 25
print(año)    # Salida: 2024

Los números enteros se pueden sumar, restar, multiplicar y dividir como se muestra en el siguiente ejemplo:


a = 10
b = 5
suma = a + b
print(suma)          # Salida: 15

resta = a - b
print(resta)         # Salida: 5

multiplicacion = a * b
print(multiplicacion) # Salida: 50

division = a / b
print(division)      # Salida: 2.0

Números decimales (float)

Los números decimales son números que tienen una parte decimal. Se pueden representar utilizando el tipo float.

También se pueden sumar, restar, multiplicar y dividir.


altura = 1.75
peso = 70.5
imc = peso / (altura ** 2)
print(altura)  # Salida: 1.75
print(peso)    # Salida: 70.5
print(imc)     # Salida: 23.02

En el caso de la división, el resultado es un número decimal, incluso si el resultado es un número entero.

También podemos obtener el residuo de una división con el operador `%`:


residuo = 10 % 3
print(residuo)  # Salida: 1

Cadenas (string)

Las cadenas son secuencias de caracteres que se pueden almacenar en variables. Se definen utilizando comillas simples o dobles.


nombre = "Robert"
edad = '36'
print(nombre, edad)  # Salida: Robert 36

Hay muchas operaciones que podemos hacer con los strings en Python. Verémos esto en mas detalle en un par de semanas.

Por ejemplo, podemos concatenar strings:


saludo = "Hola, " + nombre
print(saludo)  # Salida: Hola, Robert

Podemos convertir un número a string:


edad = 25
print("Tengo " + str(edad) + " años")  # Salida: Tengo 25 años

Y si un string es un numero, podemos convertirlo a un número entero o decimal:


numero = "25"
print(numero.isdigit())  # Salida: True
print(int(numero) + 5)  # Salida: 30
print(float(numero) + 5.5)  # Salida: 30.5

para acceder a un caracter específico de un string, usamos los indices. Los indices empiezan en 0, y los indices negativos empiezan desde la derecha.:


print(nombre[0])  # Salida: R
print(nombre[1])  # Salida: o
print(nombre[-1])  # Salida: t

Listas (list)

Las listas son colecciones de elementos que se pueden almacenar en una variable. Se definen utilizando corchetes [] y se separan con comas.


frutas = ["manzana", "banana", "naranja"]
print(frutas)  # Salida: ['manzana', 'banana', 'naranja']

Usamos los `indices` para acceder a los elementos de una lista.

 
print(frutas[0])  # Salida: manzana
print(frutas[1])  # Salida: banana
print(frutas[1])  # Salida: banana
print(frutas[-1])  # Salida: naranja

Si tienes un archivo de textos y quieres segmentarlo en oraciones, puedes almacenarlas en una lista. También son útiles las listas cuando queremos segmentar una frase en palabras. usamos el método `split()`


texto = "Este es un ejemplo de texto. Este es otro ejemplo."
oraciones = texto.split(".")
print(oraciones)  # Salida: ['Este es un ejemplo de texto en Python', ' Este es otro ejemplo', '']

palabras = oraciones[0].split(" ")
print(palabras)  # Salida: ['Este', 'es', 'un', 'ejemplo', 'de', 'texto', 'en', 'Python']

Podemos agregar elementos a una lista con el método `append()`, y concatenar listas con el método `extend()`


frutas.append("pera")
print(frutas)  # Salida: ['manzana', 'banana', 'naranja', 'pera']
frutas.extend(["uva", "sandía"])
print(frutas)  # Salida: ['manzana', 'banana', 'naranja', 'pera', 'uva', 'sandía']

Diccionarios (Dict)

Los diccionarios son colecciones de pares clave-valor que se pueden almacenar en una variable. Se definen utilizando llaves {} y colon : y se separan con comas.

Los diccionarios son útiles cuando queremos almacenar información que tiene una estructura clave-valor, como pares de traducciones

para acceder a un valor, usamos la clave correspondiente:


traducciones = {"apple": "manzana", "banana": "banana", "orange": "naranja"}
print(traducciones["apple"])  # Salida: manzana



persona = {"nombre": "Juan", "edad": 25}
print(persona)  # Salida: {'nombre': 'Juan', 'edad': 25}

Podemos agregar elementos a un diccionario con la sintaxis `diccionario[nueva_clave] = nuevo_valor`


persona["ciudad"] = "México"
print(persona)  # Salida: {'nombre': 'Juan', 'edad': 25, 'ciudad': 'México'}

Si intentamos agregar una clave que ya existe, el valor se actualiza:


persona["edad"] = 26
print(persona)  # Salida: {'nombre': 'Juan', 'edad': 26, 'ciudad': 'México'}

Si queremos borrar una entrada en el diccionario (o sea, una clave y su valor correspondiente), usamos `del`:


del persona["ciudad"]
print(persona)  # Salida: {'nombre': 'Juan', 'edad': 26}

Conjuntos (Set)

Los conjuntos son colecciones de elementos únicos que se pueden almacenar en una variable. Se definen utilizando llaves {} y se separan con comas.

Los conjuntos son útiles cuando queremos almacenar elementos únicos, como colores o palabras, y averiguar la pertenencia de un elemento a un conjunto.


colores = {"rojo", "azul", "verde"}
print("rojo" in colores)  # Salida: True

Podemos convertir una lista a un conjunto con la función `set()`:


numeros = [1, 2, 3, 4, 5, 1, 2, 3]
numeros_unicos = set(numeros)
print(numeros_unicos)  # Salida: {1, 2, 3, 4, 5}

Booleanos (Bool)

Los booleanos son valores que representan verdadero o falso. Se pueden representar utilizando los valores True y False.

Las siguientes expresiones nos devolverán valores booleanos:

 

5 > 3  # Salida: True
"orale" == "órale" # Salida: False
100 == (10*10) # Salida: True

None

None es un valor especial en Python que representa la ausencia de un valor. Se puede utilizar para inicializar variables o para verificar si una variable tiene un valor.
